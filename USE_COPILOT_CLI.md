# Using Copilot CLI for Project Management

## Copilot CLI is Already Installed! ✅

Version: 0.0.356

## How to Use Copilot CLI

### Launch Copilot CLI
```bash
copilot
```

### Ask It to Move Issues

Once in Copilot CLI, you can ask:

```
Move issues 1, 4, 5, 6, 7 from "To triage" to "Ready" status in the "Production Website" project
```

Or:

```
How do I move GitHub issues between project board columns using the GitHub API?
```

### Copilot CLI Commands

- `/login` - Authenticate with GitHub
- `/model` - Choose AI model
- `/feedback` - Submit feedback
- `/banner` - Show animated banner

## What Copilot CLI Can Help With

- ✅ Writing scripts to move issues
- ✅ Understanding GitHub API for projects
- ✅ Creating automation workflows
- ✅ Explaining project board structure

## Note

Copilot CLI requires:
- Active GitHub Copilot subscription (Individual, Business, or Enterprise)
- Authentication with GitHub
- Project permissions (if moving issues)
